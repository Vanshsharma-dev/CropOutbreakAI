## Problem
Farmers suffer major crop losses due to late detection of plant diseases and lack of early outbreak warnings.

## Solution
CROPOUTBREAK.ai is an AI-powered system that detects crop diseases from leaf images and provides early outbreak risk indications to help farmers take timely action.

## Features
- Image-based disease detection
- AI confidence score
- Early outbreak warning
- Farmer-friendly interface

## Technologies Used
- Python
- PyTorch
- Streamlit
- Computer Vision
